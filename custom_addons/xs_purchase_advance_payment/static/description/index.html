Purchase Advance Payment

This module can help you make an advance payment on Purchase Orders. In some cases, vendor may need to pay advance payment.

Advance Payment

Select the payment method and input advance amount.

Advance Payment Lines

After making advance payment, it will generate entry in Advance Payment tab.

Vendor Bills

After creating bill, you would see outstanding debits for the particular supplier.

Vendor Bills

You can add the oustanding debits for that supplier, which you had taken as advance payment.

Register Payment

Register the remaining payable amount for that supplier.
